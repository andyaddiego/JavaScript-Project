DrinkScript es una simulacion interactiva de una tienda de bebidas, es mi entrega final del curso realizado en Coderhouse.
No cuenta con diseño responsivo, hecho con HTML5 y CSS3 manual, dentro de lo que aprendí de forma autodidacta.

Mejoras de UX en caso que fuese un proyecto que saliese a produccion:

- Búsqueda: 
	- Ocultar el titulo de las secciones las cuales no contienen items dentro de la búsqueda.
	
- Carrito: 
	- Corrección de aspectos visuales. 
	- Botones para icrementar/disminuir la cantidad de cada producto.

- Newsletter:
	- Evento submit: Incorporado para una preentrega, no cuenta con localStorage ni alerta al clickear.

- Newsletter:
